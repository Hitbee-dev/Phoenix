## Dependency (for flask-server)
- Common
```
    pip install flask
```

- Crawling
```
    pip install bs4
    pip install requests
    pip install selenium==4.4.0
    pip install webdriver_manager
    pip install openpyxl
```

- DB set
```
    pip install pymysql
```

- Summarize, QnA Model 
```
    pip install torch
    pip install transformers
```
