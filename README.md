# Stack
- **Front-End**
    - **Application**:
        - Framework: `Flutter`
        - Language: `Dart`
    - **Chrome Extension**:
        - Framework: `Jquary`
        - Language: `HTML`, `CSS`, `JavaScript`
- **Back-End**
    - **Cloud Service:** Azure
        - **http Protocol Server(Application)**:
            - Framework: `Flask`
            - Language: `Python`
        - **https Protocol Server(ChromeExtension)**:
            - Framework: `Django`
            - Language: `Python`
        - **Crowling**:
            - Language: `Python`
            - Module: `BeautifulSoup, Selenium`
        - **Database**: `MySQL`
- **Model**
    - **News Summary Model**:
        - Framework: `Pytorch`
        - Language: `Python`
        - Model: `KoBART`
        - Datasets: `Dacon 한국어문서생성요약 AI 경진대회`
    - **Quiz Create Model**:
        - Framework: `Pytorch`
        - Language: `Python`
        - Model: `T5`
        - Datasets: `KorQuAD`
    - **News Selection:**
        - Clustering: `DBSCAN`
        - Language: `Python`
